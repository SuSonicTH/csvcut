Usage: csvcut [OPTION]... [FILE]...

filter csv files by column name(s)
With no FILE, or when FILE is -, read standard input.

Options:
        -h, --help               display this help and exit
        -v, --version            display version/license and exit
        -l, --listHeader         print the file header and exit

    input:
        -t, --tab                use tab (\t) as a field separator
        -c, --comma              use comma (,) as a field separator (default)
        -s, --semicolon          use semicolon (;) as a field separator
        -p, --pipe               use pipe (|) as a field separator
        -d, --doubleQuote        use double quote (") for quoting
        -q, --quote              use single quote (') for quoting
            --noQuote            don't use quoting (default)
        -h, --header             use given heder (field1,field2,...)
        -n, --noHeader           file has no header

    output formats:
        --format Csv             csv file for format options see output section
        --format LazyMarkdown    compact markdown table format NOT formatted

    output:
        -T, --outputTab          use tab (\t) as field separator
        -C, --outputComma        use comma (,) as field separator (default)
        -S, --outputSemicolon    use semicolon (;) as field separator
        -P, --outputPipe         use pipe (|) as output field separator
        -D, --outputDoubleQuote  use double quote (") for output quoting
        -Q, --outputQuote        use single quote (') for output quoting
            --outputNoQuote      don't use output quoting (default)
        -N, --outputNoHeader     don't write header to output
        
    processing:
        -F, --fields            comma separated list of field names to output
                                can be used multiple times
                                
        -I, --indices           comma separated indices of fields to output
                                can be used multiple times
            --filter            list of filters that are applied
                                argument field1=value1,field2=value2,....
            --trim              removes space & tab characters from start & end
